# Cudy tr3000 immortalwrt-24.10 带闭源硬件加速
## 使用说明：
### Uboot：  
[cudy tr3000 v1中文三分区DHCP uboot第二版](https://www.right.com.cn/forum/thread-8415351-1-1.html)  
使用mod-112m刷入即可  
### 固件特点：  
纯净、简洁、追随源码更新  
### 添加插件：  
1、USB模块支持，移动硬盘，同添加了usb-net支持  
2、网络共享，ksmbd  
3、MTK硬件网络加速
